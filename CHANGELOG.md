# 变更日志 / Changelog — FoodSafeML (foodsafeml)

所有版本均遵循「不编造」原则：缺失的限量值 / 多语字段一律以 `[待填写]` 标注，权威数据均附来源 URL 可追溯。

---

## [1.12.0] — 2026-07-12 · 跨 Agent 通用·技能解耦

- **解耦 WorkBuddy 私有耦合，改为跨 Agent 通用技能**：`SKILL.md` / `README.md` 去除 `~/.workbuddy/` 路径与私有工具名（`WebSearch` / `WebFetch` / `AskUserQuestion` / `Agent` / `minimax-xlsx`），统一改为「按所在平台替换」的能力化描述；新增「跨 Agent 通用说明」与 README「各智能体平台中使用」小节，给出 WorkBuddy / Claude Code / Cursor / OpenAI Codex / 通用 LLM+工具链 的加载与调用指引。
- **新增本文件 `CHANGELOG.md`**，作为跨平台可读的版本索引（覆盖 v1.7.0 → v1.12.0）。
- **确认 Python 生成器已跨平台**：`scripts/build_xlsx.py` 仅依赖 `os/re/json/sys/openpyxl`，`sys.argv` 传参，无任何平台 SDK 耦合；在任意装有 Python 3.10+ 与 openpyxl 的环境可直接运行 `python build_xlsx.py <食品目录> [食品名]`。
- **修正旧版附录八过期数字**：「5 大类 72 项」→「7 大类 126 项」；「约 27% / 29% 单元格 `[待填写]`」→「约 0.7%（实测 0.69%，含用户数据）」。
- 版本 1.11.0 → 1.12.0；`SKILL_VER` 与示例产物「skill 版本」单元格同步更新。

## [1.11.0] — 2026-07-12 · 国际基准回填·指标压实

- `BASE_LIMITS` 由 16 → 133 条（Codex / EU / US / JP / CN 五套基准，真实联网检索，每条附来源 URL 与置信度；Codex 干制水果 SO₂ 据官方原文由 ≤1500 修正为 ≤1000 mg/kg）。
- 回填逻辑升级（`resolved_cell`）：参照 Codex / EU / US / JP 的地区直接回填**绿色具体限量值 + 外链法规原文**（原版仅显示黄色「适用标准框架」文字）；自有 / 国际地区改为黄色框架引用。
- `[待填写]` 空白率 27.1% → **0.69%**（含用户数据）/ 0.97%（纯基准）。

## [1.10.0] — 2026-07-11 · 全量指标扩展·理化/微生物/感官全面覆盖

- `ALL_INDICATORS` 由 73 → 126 项：新增「理化指标」17 项、「感官指标」10 项，扩充污染物（13→27）、真菌毒素（7→12）、微生物/致病菌（13→20）。
- 权威限量补强：新增 Codex CXS 193-1995 与 EU (EU) 2023/915 对铅/镉/苯并[a]芘的限量（附 eur-lex/fao URL）。

## [1.9.0] — 2026-07-08 / 11 · 全量地区 + 权威数据 + 标准体系映射

- 地区清单 36 → **235** 个（独立 `scripts/regions_data.py`，按洲分组：国际/亚洲/欧洲/北美/南美/大洋洲/非洲）。
- 新增 `BASE_LIMITS` 权威限量库 + `resolved_cell()` 标准体系映射回填；`[待填写]` 占比 93.8% → 29.3%。
- `precise-search` 补强：57 条干制水果 / 芒果品类数据（14 条高置信数值 + 43 条如实标记「未设定」），修正赭曲霉毒素A(EU)、SO₂(EU) 等，新增总黄曲霉毒素指标（73→73 实际净增 1，真菌毒素类）。

## [1.8.0] — 2026-07-07 · 全量矩阵架构

- 生成器由「数据驱动」升级为「指标驱动 × 地区驱动」全量矩阵：`ALL_INDICATORS`（72 项）+ `ALL_REGIONS`（36 个）。
- Sheet3 全量遍历，Sheet4 = 72 × 36 矩阵；新增 `source_map` 机制。

## [1.7.1] — 2026-07-08 · 生成器修复

- 修复 3 处缺陷：配料多语对照仅渲染固定字段、基础信息缺「配料」整行、国际/英文名与别名/同义词重复；补 `GB 2763-2026` 英文映射。

## [1.7.0] — 2026-07-07 · 工作簿结构变更

- Sheet1 合并多语；Sheet3 由 25 列扩至 31 列（新增指标名四语、法规名三语、「声称执行标准」）；
- Sheet4 改为公式透视（HYPERLINK 跳转源表）；农残逐物质展开、污染物展开为铅(Pb)/镉(Cd)/汞(Hg)/砷(As)；强制保留微生物/致病菌指标。
